COORDINATE MAPPING:
  Function: minimap_to_map_coords(minimap_x, minimap_y, width, height)
  
  Steps:
    1. Normalize minimap coords to 0-1 range
       norm_x = minimap_x / minimap_width
       norm_y = minimap_y / minimap_height
    
    2. Scale to map dimensions
       map_x = int(norm_x * 560)
       map_y = int(norm_y * 420)
    
  Result: Perfect alignment with Theia Sky Ruins map

CREEP DETECTION:
  HSV Range: [15, 60, 60] to [40, 255, 200]
  Method: Hough Circle Detection
  Radius: 3-10 pixels
  Verification: Must have dark yellow pixels inside
  
  Clustering: Round to nearest 5 pixels
    key = (round(x / 5) * 5, round(y / 5) * 5)

OBJECTIVE DETECTION:
  HSV Range: [15, 100, 180] to [40, 255, 255]
  Method: Contour analysis
  Size: 50-500 pixels area
  Shape: Non-circular (Pokemon face shape)
  
  Timer Exclusion: Skip if in top 20% or bottom 20% of minimap

UPTIME CALCULATION:
  For each position (x, y):
    1. Count frames where detected
    2. Each frame = 1 second
    3. Convert: minutes = count // 60, seconds = count % 60
    4. Format: f"{minutes:02d}:{seconds:02d}"

HEATMAP GENERATION:
  Purple Heatmap:
    1. Accumulate all purple positions in 2D array
    2. Apply Gaussian blur (35x35 kernel)
    3. Normalize to 0-1 range
    4. Blend with map: result = map * (1-alpha) + purple * alpha
  
  Orange Heatmap:
    1. Same process as purple
    2. Blend on top of purple layer
  
  Uptime Overlays:
    1. Draw circles at camp/objective positions
    2. Add MM:SS text labels
    3. White outline + black fill for readability
